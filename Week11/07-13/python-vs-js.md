# Python vs JS

## no semicolons in Python
```
n = 1
```

## no curly braces to denote code blocks, use a colon and indentation
```
if n > 1:
  print 'greater than 1'

for n in numbers:
  print n

def add(x, y):
  return x + y
```

## plain objects in JS are dictionaries
JS object literal
```
var student = {
  name: 'Kyle',
  score: 100
}
```
## in python...
Python dictionary
```
student = {
  'name': 'Kyle',
  'score': 100
}
```
